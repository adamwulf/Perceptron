PART 2: PERCEPTRON ALGORITHM
================================================================================

The perceptron is a single-layer neural network that classifies binary patterns.

FORMULA:
  OUTPUT = SUM( Switch(i) * Weight(i) ) + Bias

Where:
  - Switch(i) = +1 if ON, -1 if OFF (not 0/1!)
  - Weight(i) = dial value from -30 to +30
  - Bias = bias dial value from -30 to +30
  - OUTPUT = real number (displayed on analog meter)

CLASSIFICATION:
  - Positive output = one class (e.g., "T" shape)
  - Negative output = other class (e.g., "J" shape)

LEARNING RULE (Perceptron Learning Algorithm):
When pressing [Learn +] (want POSITIVE output) AND output is currently wrong:
  - For each ON switch: weight[i] += learningRate
  - For each OFF switch: weight[i] -= learningRate
  - Bias += learningRate

When pressing [Learn -] (want NEGATIVE output) AND output is currently wrong:
  - For each ON switch: weight[i] -= learningRate
  - For each OFF switch: weight[i] += learningRate
  - Bias -= learningRate

IMPORTANT: If output is already correct sign, NO changes are made.
All weights clamped to range [-30, +30].

PART 5: CUSTOM CONTROLS SPECIFICATIONS
================================================================================

All controls use GDI+ for rendering with SmoothingMode.AntiAlias.
All controls inherit from Control and use ControlStyles for double buffering.

----- SwitchControl -----
Size: Default 50x80 (scalable for larger grids)
Components:
  - LED indicator at top (red when ON, dark when OFF)
  - LED has glow effect when ON (alpha 40 red glow)
  - Toggle track (dark gray rounded rectangle)
  - Toggle handle (moves up/down based on state)
  - Toggle color: ON = Color.FromArgb(100, 100, 100), OFF = Color.FromArgb(70, 70, 70)
  - Border brightens on hover
Behavior:
  - Clicking toggles state
  - Space bar toggles when focused
  - StateChanged event
  - Value property: +1 when ON, -1 when OFF

----- KnobControl -----
Size: Default 70x90 (scalable)
Range: -30 to +30
Components:
  - Circular knob with pointer indicator
  - Tick marks around circumference
  - Major ticks at -30, 0, +30
  - Value display below knob
  - Metallic gradient (dark gray tones)
  - Subtle glow/highlight at top
Behavior:
  - Click to select (becomes "active" knob)
  - Arrow keys change value when active
  - Mouse drag rotates knob
  - Step size: 0.05 (fine control)
  - Zero points straight UP
  - ValueChanged event

----- SettingsKnobControl -----
Similar to KnobControl but with configurable range.
Properties:
  - MinValue, MaxValue, Step
  - TickValues, MajorTickValues
  - MinValuePointsUp (if true, min value points up; if false and symmetric range, zero points up)
  - ValueFormat (e.g., "0", "0.00")
Used for: Grid Size (1-10), Learning Rate (-30 to +30)

----- AnalogMeterControl -----
Size: 180x130
Range: -100 to +100
Components:
  - Circular arc gauge
  - Needle that rotates based on value
  - Graduation marks
  - Metallic bezel around edge
  - Dark face with light markings

----- OutputLedControl -----
Large LED indicator
Components:
  - Circular LED (green when on, dark when off)
  - Glow effect when ON using PathGradientBrush
  - "OUTPUT" label below

----- MechanicalPushButton -----
Two variants: IsSquare = true (rectangular) or IsSquare = false (round)
Size: Approximately 50x55
Components:
  - Metal plate label above button with text
  - Four corner divots (screw holes) on label plate
  - Button with recessed housing (dark)
  - Button bezel (gradient)
  - Button face (gradient, darker when pressed)
  - Glow effect inside button using PathGradientBrush
Glow Colors:
  - BRAIN, MANUAL: Yellow glow Color.FromArgb(255, 220, 80)
  - SAVE: Conditional - green when weights dirty, none otherwise
  - LOAD: Faint red Color.FromArgb(200, 80, 60)
  - LEARN+: Conditional - green when any switch ON, none otherwise
  - LEARN-: Conditional - yellow when any switch ON, none otherwise
Behavior:
  - Visual press state on mouse down
  - ButtonClick event on mouse up

----- ArrowButton -----
Triangular directional buttons (D-pad style)
Directions: Up, Down, Left, Right
Size: Up/Down = 40x16, Left/Right = 16x40
Components:
  - Triangular shape pointing in direction
  - Pale yellow glow effect
  - Metallic gradient
Behavior:
  - Clicking shifts the switch pattern in that direction
  - Patterns shift and wrap out (bits that shift off disappear)

----- MetalLabelControl -----
Metal plate with engraved text
Components:
  - Linear gradient background (dark greens/grays)
  - Brushed texture (faint horizontal lines)
  - Beveled edges (light top, dark bottom)
  - Four corner screws/divots
  - Engraved text (shadow + light text)
Properties:
  - LabelText: The text to display
  - Subdued: When true, uses more muted colors
  - CustomTextColor: Optional override for text color (used for video link = green)

----- MetalPlateControl -----
Larger instruction plate for multi-line text
Components:
  - Similar metallic styling to MetalLabelControl
  - Supports multiple lines of text (InstructionLines array)
  - Text appears engraved

----- FormulaPlateControl -----
Displays the perceptron formula
Shows: "Output = Σ(Switch × Weight) + Bias"

PART 8: DYNAMIC SCALING
================================================================================

When grid size changes (1-10):
  - Switches and knobs scale down for larger grids (7x7+)
  - Formula for switch size at 7+: Math.Max(28, 50 - (gridSize - 6) * 7) width
  - Formula for knob size at 7+: Math.Max(45, 70 - (gridSize - 6) * 8) width
  - Window auto-grows to fit grid
  - At 8x8+, shift knob panel slightly right for better fit
  - At 8x8+, reduce output panel width

1. GraphicsExtensions: Create extension methods FillRoundedRectangle and DrawRoundedRectangle for Graphics class.

5. Learn button glow: Only show glow when at least one switch is ON.

6. Save button glow: Only show green glow when _weightsDirty flag is true.

8. Manual chapters use Roman numerals (I, II, III, IV, V, VI, VII, VIII, IX, X, XI).

9. Default learning rate is 10.0, not 1.0.
